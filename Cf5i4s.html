문제

입력받은 달의 영어이름을 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행순서를 클릭하세요 참고로, 달의 영어이름은 
1월 - January, 
2월 - February, 
3월 - March, 
4월 - April, 
5월 - May, 
6월 - June, 
7월 - July, 
8월 - August, 
9월 - September, 
10월 - October, 
11월 - November, 
12월 - December와 같습니다.
1월 ~ 12월 사이의 월을 입력하세요: 3
3월은 March입니다

프로그램 코드

	#include <stdio.h>

		

1	int main() {

		printf( "1월 ~ 12월 사이의 월을 입력하세요: " );

		int month = 0;

2		scanf( "%d", &month );

		

3		switch ( month ) {

4			case 1:

5				printf( "1월은 January입니다" );

6				break;

7			case 2:

8				printf( "2월은 February입니다" );

9				break;

10			case 3:

11				printf( "3월은 March입니다" );

12				break;

13			case 4:

14				printf( "4월은 April입니다" );

15				break;

16			case 5:

17				printf( "5월은 May입니다" );

18				break;

19			case 6:

20				printf( "6월은 June입니다" );

21				break;

22			case 7:

23				printf( "7월은 July입니다" );

24				break;

25			case 8:

26				printf( "8월은 August입니다" );

27				break;

28			case 9:

29				printf( "9월은 September입니다" );

30				break;

31			case 10:

32				printf( "10월은 October입니다" );

33				break;

34			case 11:

35				printf( "11월은 November입니다" );

36				break;

37			default:

38				printf( "12월은 December입니다" );

		}

		

39		return 0;

	}
