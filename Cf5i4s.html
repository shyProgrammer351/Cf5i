문제

입력받은 달의 영어이름을 출력하는 문제입니다 실행순서를 클릭하세요 참고로, 달의 영어이름은 
1월 - January, 
2월 - February, 
3월 - March, 
4월 - April, 
5월 - May, 
6월 - June, 
7월 - July, 
8월 - August, 
9월 - September, 
10월 - October, 
11월 - November, 
12월 - December와 같습니다.
1월 ~ 12월 사이의 월을 입력하세요: 4
4월은 April입니다

프로그램 코드

	#include <stdio.h>

 

1	int main() {

2		printf( "1월 ~ 12월 사이의 월을 입력하세요: " );

		int month = 0;

3		scanf( "%d", &month );


4		switch ( month ) {

5			case 1:

6				printf( "1월은 January입니다" );

7				break;

8			case 2:

9				printf( "2월은 February입니다" );

10				break;

11			case 3:

12				printf( "3월은 March입니다" );

13				break;

14			case 4:

15				printf( "4월은 April입니다" );

16				break;

17			case 5:

18				printf( "5월은 May입니다" );

19				break;

20			case 6:

21				printf( "6월은 June입니다" );

22				break;

23			case 7:

24				printf( "7월은 July입니다" );

25				break;

26			case 8:

27				printf( "8월은 August입니다" );

28				break;

29			case 9:

30				printf( "9월은 September입니다" );

31				break;

32			case 10:

33				printf( "10월은 October입니다" );

34				break;

35			case 11:

36				printf( "11월은 November입니다" );

37				break;

38			default:

39				printf( "12월은 December입니다" );

		}


40		return 0;

	}
