문제

알파벳 소문자 단어를 입력받아 대문자 단어를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
알파벳 소문자 단어를 입력하세요: cat
소문자 cat의 대문자는 CAT입니다^^


알파벳 소문자 단어를 입력하세요: joy
소문자 joy의 대문자는 JOY입니다^^


알파벳 소문자 단어를 입력하세요: happy
소문자 happy의 대문자는 HAPPY입니다^^

알고리즘

프로그램 시작 
   알파벳 소문자 단어를 입력받기 
   알파벳 소문자 단어를 대문자 단어로 변환하기 
   입력값 출력하기 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/Alphabet.c 

	#include <stdio.h>  

	 

	// 프로그램 시작 

1	int main() 

	{ 

		// 5칸짜리 배열을 선언하고 초기화  // [주석포함] 

		char lowercase[5] = { '\0' };  

		char uppercase[5] = { '\0' }; 

2		int  index = 0; 

	 

		// 알파벳 소문자 단어를 입력받기 

3		printf("알파벳 소문자 단어를 입력하세요: " ); 

		// cat을 입력받는다고 가정 

4		scanf( "%s", lowercase );						 

	 

		// 알파벳 소문자 단어를 대문자 단어로 변환하기 

5		for ( index = 0; lowercase[ index ] != '\0'; index++ )			 

		{ 

6			uppercase[ index ] = lowercase[ index ] - 'a' + 'A';		 

		} 

	 

		// 입력값 출력하기 

7		printf("소문자 %s의 대문자는 %s입니다^^", lowercase, uppercase );	 

	 

		// 프로그램 종료 

8		return 0; 

	}
