문제

사용자가 입력한 년도가 윤년인지 평년인지 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 참고로, 4로 나누어 떨어지는 해는 윤년이지만, 
그 중에서 100으로 나누어 떨어지는 해는 평년입니다. 
그런데 400으로 나누어 떨어지는 해는 윤년입니다.
년도를 입력하세요: 2000
2000년은 윤년입니다.

프로그램 코드

	#include <stdio.h> 
1	int main() {
2		int year = 0;
3		printf( "년도를 입력하세요: " );
4		scanf( "%d", &year );
5		if ( ( year % 400 ) == 0 ) {
6			printf( "%d년은 윤년입니다.", year );
		}
7		else if ( ( year % 100 ) == 0 ) {
8			printf( "%d년은 평년입니다.", year );
		}
9		else if ( ( year % 4 ) == 0 ) {
10			printf( "%d년은 윤년입니다.", year );
		}
11		else {
12			printf( "%d년은 평년입니다.", year );
		}
13		return 0;
	}
