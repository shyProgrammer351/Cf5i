문제

사용자가 입력한 나이에 맞는 버스요금을 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
참고로, 버스요금은 다음과 같습니다. 
* 영유아(만5세 이하)는 무료 
* 초등학생(만6세 - 만12세)은 현금기준 450원 
* 청소년(만13세 - 만18세)은 현금기준 1,000원 
* 성인(만19세 이상)은 현금기준 1,300원
나이를 입력하세요: 20
버스요금은 1,300원입니다.

프로그램 코드

	#include <stdio.h> 
1	int main() {
2		int age = 0;
3		printf( "나이를 입력하세요: " );
4		scanf( "%d", &age );
5		if ( age <= 5 ) {
6			printf( "버스요금은 무료입니다." );
		}
7		else if ( ( 6 <= age ) && ( age <= 12 ) ) {
8			printf( "버스요금은 450원입니다." );
		}
9		else if ( ( 13 <= age ) && ( age <= 18 ) ) {
10			printf( "버스요금은 1,000원입니다." );
		}
11		else {
12			printf( "버스요금은 1,300원입니다." );
		}
13		return 0;
	}
