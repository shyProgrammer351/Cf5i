문제

A양과 B군이 가위바위보를 선택하면 승패를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
A양이 이겼습니다.


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
보를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
B군이 이겼습니다.


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
A양과 B군이 비겼습니다.

알고리즘

프로그램시작 
   A양이 가위바위보중 하나를 선택 
   B군이 가위바위보중 하나를 선택 
   A양과 B군의 가위바위보 승패 출력 
      먼저 A양과 B군이 같으면 비김 
      A양이 가위이고 B군이 보를 낸 경우 A양이 승리 
      A양이 바위이고 B군이 가위를 낸 경우 A양이 승리 
      A양이 보이고 B군이 바위를 낸 경우 A양이 승리 
      나머지는 B군이 승리 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/RockPaperScissors.c

	#include <stdio.h>

 

	// 프로그램시작 

1	int main() {

		int playerA = 0;

2		int playerB = 0;

	 

		// A양이 가위바위보중 하나를 선택 

3		printf( "A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " );

4		scanf( "%d", &playerA );

5		switch( playerA ) {

6		case 1:

7			printf( "가위를 냈습니다.\n" );

8			break;

9		case 2:

10			printf( "바위를 냈습니다.\n" );

11			break;

12		case 3:

13			printf( "보를 냈습니다.\n" );

		}

	 

		// B군이 가위바위보중 하나를 선택 

14		printf("B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: ");

15		scanf( "%d", &playerB );

16		switch( playerB ) {

17		case 1:

18			printf( "가위를 냈습니다.\n" );

19			break;

20		case 2:

21			printf( "바위를 냈습니다.\n" );

22			break;

23		case 3:

24			printf( "보를 냈습니다.\n" );

		}

	 

		// A양과 B군의 가위바위보 승패 출력 

		// 먼저 A양과 B군이 같으면 비김  

25		if ( playerA == playerB ) {

26			printf( "\nA양과 B군이 비겼습니다.\n" );

		}

		// A양이 가위이고 B군이 보를 낸 경우 A양이 승리 

27		else if ( ( playerA == 1 ) && ( playerB == 3 ) ) {

28			printf( "\nA양이 이겼습니다.\n" );

		}

		// A양이 바위이고 B군이 가위를 낸 경우 A양이 승리 

29		else if ( ( playerA == 2 ) && ( playerB == 1 ) ) {

30			printf( "\nA양이 이겼습니다.\n" );

		}

		// A양이 보이고 B군이 바위를 낸 경우 A양이 승리 

31		else if ( ( playerA == 3 ) && ( playerB == 2 ) ) {

32			printf( "\nA양이 이겼습니다.\n" );

		}

		// 나머지는 B군이 승리  

33		else {

34			printf( "\nB군이 이겼습니다.\n" );

		}

	 

		// 프로그램 종료 

35		return 0;

	}
