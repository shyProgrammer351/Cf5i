문제

사용자가 입력한 년도가 윤년인지 평년인지 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 참고로, 4로 나누어 떨어지는 해는 윤년이지만, 
그 중에서 100으로 나누어 떨어지는 해는 평년입니다. 
그런데 400으로 나누어 떨어지는 해는 윤년입니다.
년도를 입력하세요: 2000
2000년은 윤년입니다.


년도를 입력하세요: 2100
2100년은 평년입니다.


년도를 입력하세요: 2048
2048년은 윤년입니다.

코드 빈칸 채우기

#include <stdio.h>  
int main() { 
	int year = 0; 
	printf( "년도를 입력하세요: " ); 
	scanf( "%d", &year ); 
	 ( ( year  400 )  0 ) { 
		printf( "%d년은 윤년입니다.", year ); 
	} 
	 ( ( year  100 )  0 ) { 
		printf( "%d년은 평년입니다.", year ); 
	} 
	 ( ( year  4 )  0 ) { 
		printf( "%d년은 윤년입니다.", year ); 
	} 
	 { 
		printf( "%d년은 평년입니다.", year ); 
	} 
	return 0; 
}
