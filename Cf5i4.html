문제

입력받은 달의 영어이름을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 달의 영어이름은 
1월 - January, 
2월 - February, 
3월 - March, 
4월 - April, 
5월 - May, 
6월 - June, 
7월 - July, 
8월 - August, 
9월 - September, 
10월 - October, 
11월 - November, 
12월 - December와 같습니다.
1월 ~ 12월 사이의 월을 입력하세요: 4
4월은 April입니다 


1월 ~ 12월 사이의 월을 입력하세요: 3
3월은 March입니다 


1월 ~ 12월 사이의 월을 입력하세요: 5
5월은 May입니다

알고리즘

프로그램 시작 
   사용자에게 달을 입력받기 
   입력받은 달의 영어이름을 출력 
      1월이면 January를 출력 
      2월이면 February를 출력 
      3월이면 March를 출력 
      4월이면 April를 출력 
      5월이면 May를 출력 
      6월이면 June를 출력 
      7월이면 July를 출력 
      8월이면 August를 출력 
      9월이면 September를 출력 
      10월이면 October를 출력 
      11월이면 November를 출력 
      12월이면 December를 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/Month.c

	#include <stdio.h>

  

	// 프로그램 시작 

1	int main() {

		// 사용자에게 달을 입력받기 

2		printf( "1월 ~ 12월 사이의 월을 입력하세요: " );

		int month = 0;

3		scanf( "%d", &month );

	 

		// 입력받은 달의 영어이름을 출력 

4		switch ( month ) {

			// 1월이면 January를 출력 

5			case 1:

6				printf( "1월은 January입니다" );

7				break;

			// 2월이면 February를 출력 

8			case 2:

9				printf( "2월은 February입니다" );

10				break;

			// 3월이면 March를 출력 

11			case 3:

12				printf( "3월은 March입니다" );

13				break;

			// 4월이면 April를 출력 

14			case 4:

15				printf( "4월은 April입니다" );

16				break;

			// 5월이면 May를 출력 

17			case 5:

18				printf( "5월은 May입니다" );

19				break;

			// 6월이면 June를 출력 

20			case 6:

21				printf( "6월은 June입니다" );

22				break;

			// 7월이면 July를 출력 

23			case 7:

24				printf( "7월은 July입니다" );

25				break;

			// 8월이면 August를 출력 

26			case 8:

27				printf( "8월은 August입니다" );

28				break;

			// 9월이면 September를 출력 

29			case 9:

30				printf( "9월은 September입니다" );

31				break;

			// 10월이면 October를 출력 

32			case 10:

33				printf( "10월은 October입니다" );

34				break;

			// 11월이면 November를 출력 

35			case 11:

36				printf( "11월은 November입니다" );

37				break;

			// 12월이면 December를 출력 

38			default:

39				printf( "12월은 December입니다" );

		}

	 

		// 프로그램 종료 

40		return 0;

	}
